## 🚀 Key "WOW" Features

### 1. 🌍 Map Vitality (Hyper-Active UI)
*   **Swarm Flow**: Moving SVG particles represent real-time people flow across all stadium sectors.
*   **Danger Zones**: High-density nodes (7.0+) pulse with a red "Hazard Glow" to grab attention.
*   **Animated Navigation**: Active routes feature dynamic flow indicators and "energy" markers.

### 2. 🧠 AI Intelligence Insight
*   **Real-time Metrics**: A dedicated floating panel shows live **Crowd Load**, **Risk Level**, and **AI Confidence Score**.
*   **Autonomous Rerouting**: Proactive "AI OPTIMIZATION" toasts appear when the system finds safer/faster paths.
*   **Context-Aware Suggestions**: Smart cards suggest facilities (Food/Washrooms) based on lowest wait-times and current location.

### 3. 🚨 Emergency & Panic Response
*   **Anti-Trap Logic**: If the system detects the user lingering in a high-risk zone, it automatically triggers the **Panic Protocol**.
*   **Dramatic Overlays**: Full-screen emergency alerts with red-flashing VFX and mission-critical telemetry.
*   **Biometric Search Simulator**: Use the **"👤 Find Lost Person"** button to simulate AI-driven biometric target identification.
