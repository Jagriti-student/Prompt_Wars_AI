# 🤖 Smart Crowd Intelligence System

> AI-powered crowd monitoring and intelligent routing system for real-time stadium safety.

---

## 📌 Project Overview

Smart Crowd Intelligence System is an AI-based simulation that helps manage stadium crowd safety using real-time density detection and intelligent routing.

The system continuously monitors crowd movement, predicts potential risk zones, and automatically suggests safer routes.

---

## 🏆 Demo Highlights

1. User selects a route inside the stadium
2. System simulates real-time crowd movement
3. High-density zone is detected
4. System triggers emergency alert
5. AI automatically reroutes to a safer exit

---

## 🚀 Features

- Real-time crowd simulation using swarm logic
- Intelligent risk detection system
- Automatic route optimization
- Emergency alert system
- Context-aware assistant suggestions

---

## 🛠️ Tech Stack

- React 18 + Vite
- JavaScript
- CSS3 Animations (VFX)
- Custom Simulation Engine

---

## ⚙️ How It Works

1. User enters the system
2. Crowd simulation engine generates movement
3. Density levels are calculated
4. AI logic detects risk zones
5. Google Maps API generates optimal route
6. System updates UI dynamically

---

## 🔗 Google Services Integration

- 🧠 Google Gemini API – AI-based decision making and prediction
- 🗺️ Google Maps API – route optimization and navigation
- ☁️ Firebase – authentication and real-time data handling
